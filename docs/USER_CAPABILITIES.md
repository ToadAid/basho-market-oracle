# User Capabilities

This is a short user-facing summary of what the current system can do.

## Chat and guidance
- Answer crypto and trading questions in chat
- Explain technical indicators, strategy logic, and portfolio state
- Use a menu-driven Telegram experience for common actions

## Market research
- Check live token prices
- Search tokens and contracts
- Fetch market/news context
- Summarize daily alpha and sentiment
- Track whale activity and smart money behavior

## Technical analysis
- Analyze market trends
- Generate professional indicators
- Check market structure
- Run multi-timeframe confirmation
- Build swing-trade setups
- Analyze chart images

## Portfolio and trading
- Create and manage paper trading accounts
- Simulate buys and sells
- Show portfolio status and trade history
- Calculate position sizing and Kelly-style risk
- Review strategy performance and backtests

## Wallet and on-chain tools
- Create or inspect an agent wallet
- Show wallet addresses and balances
- Transfer tokens
- Get swap quotes
- Execute supported swaps
- Run on-chain token risk checks

## Alerts and monitoring
- Set smart alerts for price, sentiment, and wallet activity
- List and remove alerts
- Run recurring Forge watchlists for tracked assets

## Forge / prediction suite
- Generate token forecasts
- Record live predictions
- Evaluate due predictions
- Show prediction ledger summaries
- Add, list, delete, and run watches
- View latest Forge alerts

## Memory and continuity
- Read and write strategy files
- Keep long-term guidance in the wisdom ledger
- Anchor mission briefing and domain-specific operating context

## Important limitations
- This is research and operations support, not guaranteed financial advice.
- Some outputs are narrative summaries and should be checked against raw tool data.
- Provider auth and runtime state still matter; after major auth or code changes, a restart may be needed.
